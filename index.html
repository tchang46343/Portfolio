Automotive Quiz App

Car Enthusiast is a program that tests your knowledge about the basic mechanics of a gasoline powered car. The program allows the user to go through a series of questions, and tells the user if their answer was right or wrong. Finally, the program brings the user to a result page where after finishing all the questions they get their results on whether they passed the test. Using HTML5 and CSS to lay the foundation of the program we have transition from idea, to proof of concept. Furthermore, this program functionality really comes to life by implementing Javascript and JQuery Library making for a interactive and responsive program.
